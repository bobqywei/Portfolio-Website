Embedded and Data Analysis Algorithms

  • Developed Python scripts for research and experimental purposes at Brock University, specifically in the Physics Department under the supervision of Dr. Thad Harroun

  • Automated data collection through MEMS accelerometers, ceramic piezoelectric sensors, and oscilloscopes, improving the efficiency of such procedures. Python was also used to configure such instruments through serial

  • Implemented frequency analysis (using fast fourier transforms) for input from various piezoelectric sensors, allowing for the removal of unwanted frequencies and noise

  • Also worked with circuitry for conditioning and amplifying voltages from various input sensors in order to captured effectively by oscilloscopes

  • My research ultimately received Bronze Medal honors at the Canada Wide Science Fair and scholarship recognition from The University of Ottawa and Western University

Raspberry Pi Coding Assistant

  • Designed and developed a fully integrated voice assistant for the Raspberry Pi and tailored specifically for the needs of those learning to program in C

  • Voice recognition and translation were handled using the Watson API and requests to the service were made using Python

  • Capablities for google searching and obtaining code snippets off the internet are also made possible through Python scripts. For instance, example code related to various programming concepts and algorithms can be pulled and saved with a single voice command

  • Project was developed as a team of three. My teammates tackled much of the Java-based functionality including the editing of C files and gcc compiling with voice commands

Image Inpainting

  • Developed and trained a model capable of filling in irregular holes in an image through a single forward pass. Irregularly shaped masks can be filled in within 3 seconds of single-threaded computation on a CPU.

  • Constructed PyTorch model based on U-Net encoder-decoder architecture and the partial convolutional layers detailed in Liu, G., Reda, F.A., Shih, K.J. (2018) “Image Inpainting for Irregular Holes Using Partial Convolutions”.

  • Currently working on improvements such as compressing the weights for potential integration on mobile platforms and in-place batch-normalization for decreased GPU VRAM usage during training.
